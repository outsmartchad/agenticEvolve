# Original Vision Prompt — agenticEvolve (Personal Closed-loop Agentic System)

> Saved: 2026-03-11

## The Prompt

I want to build a personal agentic loop workflow system to read github, x, wechat, discord, whatsapp, and hacker news on tech updates and conversations about how they build personal ai tools like we're building that the person is evolving with ai everyday, that basically when we see something useful for our system, something that can boost our productivity on building cool projects, we build it as claude skills to better our future development, the vision is that the future of companies will only hire someone that has a closed-source agentic workflow system that superpower/supercharge the person who is using it and evolving every single day.

## Key Terminology (as of 2026-03)

- **Inner loop / Outer loop** — Anthropic's framing. Inner = agent task execution. Outer = orchestration, CI/CD, cross-session learning.
- **Closing the loop** — Agent self-verifies before returning. Next frontier: spontaneous verification.
- **Personal AI infrastructure** — Your full stack of agents, memory, tools, workflows.
- **AI exoskeleton** — AI as amplification of the individual, not a coworker.
- **Cognitive infrastructure** — Foundation layer for knowledge + decision flow.
- **Agent stack** — Inner loop + outer loop + memory + tools.

## Core Vision

A **Personal Closed-loop Agentic System** that:
1. Ingests signals from multiple platforms (GitHub, X, Discord, WeChat, WhatsApp, HN)
2. Filters for what's useful to YOUR system and workflow
3. Auto-builds Claude Code skills from discovered patterns
4. Compounds daily — every day the system is better than yesterday
5. Becomes your competitive moat — closed-source, personalized, evolving
